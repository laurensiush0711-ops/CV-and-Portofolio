
# Laurensius Haryo | Portfolio & CV

A modern, interactive professional portfolio and CV for a Game Designer & QA Junior transitioning to Data Analytics. Built with React, Vite, and Google Gemini AI.

## Features
- **AI Career Bot**: A Gemini-powered assistant that explains the career transition.
- **Interactive Skills Radar**: Visual representation of technical competencies.
- **Project Case Studies**: Detailed breakdowns of data and design projects.
- **Responsive Design**: Built with Tailwind CSS for all devices.

## How to Run
1. Install dependencies: `npm install`
2. Set your `VITE_API_KEY` in a `.env` file.
3. Start the development server: `npm run dev`
